###### **Kick Chat Overlay**



A lightweight desktop app that displays your chat stream live. Ideal for streamers who want floating chat without browser windows.



###### **Features**



* **Live Kick chat display**. Connect with your channel ID and see messages in real time.
* **Live TikTok chat display**. Connect to TikTok LIVE streams and see chat messages, gifts, follows, and subscriptions.
* **Dual platform support**. Connect to both Kick and TikTok simultaneously.
* **Transparency control**. Adjust the overlay visibility from fully opaque to invisible.
* **Message fade timer**. Set how long old messages stay on screen before disappearing.
* **Sound effects**. Enable notification sounds for incoming messages.
* **Click through mode**. Lock the overlay so it is visible but does not block your mouse.
* **Manual overlay test**. Preview the overlay before connecting.
* **Color customization**. Change design colors for better readability.



###### **How to Use**



* Enter your Kick channel ID and/or TikTok username.
* Click Connect to start receiving chat messages.
* Adjust transparency, fade time, and colors to your preference.
* Lock the overlay if you want it visible without blocking your screen.
* You can connect to both platforms simultaneously!



###### **Requirements**



* Windows PC
* Java runtime installed



###### **TikTok Features**



* **Chat messages** - All comments appear in the overlay with color-coded usernames
* **Gifts** - Gift notifications with emoji indicators (🎁)
* **Follows** - New follower notifications (💚)
* **Subscriptions** - Subscriber alerts (⭐)
* **Role colors** - Moderators (green) and subscribers (pink) have distinct colors
* **System messages** - Connection status and live stream events
