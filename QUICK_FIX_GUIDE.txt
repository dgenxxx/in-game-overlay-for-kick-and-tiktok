📋 BEFORE YOU START:
   1. Run: diagnose_tiktok.bat
   2. Make sure TikTok user is LIVE (critical!)
   3. Have console window visible

🚀 HOW TO TEST:
   1. Double-click: run_debug.bat
   2. Enter TikTok username (no @ symbol)
   3. Click Connect
   4. Watch console for messages

✅ SUCCESS LOOKS LIKE:
   ========================================
   TikTokClient: Successfully connected to @username
   ========================================
   
   Then when chat happens:
   TikTokClient: Comment from User: Hello!

❌ COMMON ERRORS:

   "User is not currently LIVE"
   → User must be streaming RIGHT NOW
   → Try a different user who is LIVE

   "TikTok library not installed"
   → Run setup_tiktok.bat
   → Check lib/ folder for JAR files

   "Connection hangs"
   → Check internet connection
   → Try different network
   → TikTok may be blocking your IP

   "User not found"
   → Check username spelling
   → Remove @ symbol
   → Verify user exists
